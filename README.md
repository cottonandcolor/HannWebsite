# Hannah Mara Real Estate Website

Responsive one-page real estate marketing site for Hannah Mara of Full Circle Real Estate.

## Preview locally

```bash
python3 -m http.server 8765 --bind 127.0.0.1
```

Open http://127.0.0.1:8765/

## Before publishing

1. Replace stock stand-in photos in `assets/` with Hannah’s approved headshot, Full Circle logo file, and real property photographs.
2. Confirm FormSubmit email: the contact form posts to `hannah@fullcircle-realestate.com` (first submission may require email confirmation).
3. Swap the Google reviews search link for Hannah’s Google Business Profile URL when available.
4. Verify Active / Sold / Representative Work status labels match current inventory.
