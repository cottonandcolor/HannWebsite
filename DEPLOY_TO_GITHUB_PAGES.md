# Deploy this website with GitHub Pages

## Recommended repository name

`hannah-mara-real-estate`

## Upload and publish

1. Sign in to GitHub and create a new **public** repository.
2. Open the repository and choose **Add file → Upload files**.
3. Upload the contents of this folder, including `index.html`, `assets`, and `.nojekyll`.
4. Commit the files to the `main` branch.
5. Open **Settings → Pages**.
6. Under **Build and deployment**, choose **Deploy from a branch**.
7. Choose the `main` branch and the `/ (root)` folder, then save.
8. GitHub will show the published website address in the Pages section.

For a project repository, the address normally follows this format:

`https://YOUR-USERNAME.github.io/hannah-mara-real-estate/`

For a main personal website at `https://YOUR-USERNAME.github.io/`, name the repository exactly:

`YOUR-USERNAME.github.io`

## Before public launch

- Replace the temporary headshot and logo placeholders with the approved files.
- Replace all property image placeholders with real property photos.
- Confirm current listing status and property facts.
- Add the required TREC and brokerage disclosure links.
- Confirm the email address and phone number shown on the site.
